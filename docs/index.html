Features

Everything your small team needs.

No account required, no subscription, no cloud dependency. Just install and start managing your project.

Kanban Board

Drag and drop tasks across five status columns: Backlog, Todo, In Progress, Review, Done. Full keyboard and mouse support.

Release Tracking

Group tasks into releases, track progress with visual indicators, and see at a glance which features ship in each version.

Rich Notes

Block-based editor powered by BlockNote. Hierarchical note trees with tags, formatting and image support.

End-to-End Encryption

All synced data is AES-GCM encrypted on your device before it leaves. The server only sees opaque ciphertext. Your data, your keys.

Offline-First

Everything stored locally in SQLite. Works without internet, syncs automatically when your PocketBase server is reachable.

Team Collaboration

Assign tasks to team members, share a workspace via a common space key, and sync changes in real-time across all devices.

Auto-Updates

The NSIS installer includes built-in auto-update support. Get notified of new versions and update with one click.

Native Performance

Built with Tauri v2 — no Electron, no Chromium. Tiny installer (~8 MB), fast startup, low memory footprint.

Self-Hosted Sync

Your server, your data.

Optional sync via PocketBase. Deploy in seconds with Docker, no vendor lock-in.

1

Deploy PocketBase

Run a single Docker container. No external database, no complex setup.

docker compose up -d --build
2

Create Admin

Create a superuser account via CLI or the built-in admin dashboard.

docker exec dragonfly-pb /pb/pocketbase superuser create EMAIL PASS
3

Connect & Sync

Enter your server URL, admin credentials and a shared space key in DragonFly. Collections are created automatically.

Settings → Sync → Setup Server → Connect
